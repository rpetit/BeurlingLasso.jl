# BeurlingLasso.jl

Numerical solvers for the Beurling LASSO
<br/>

## Available algorithms
* Sliding Frank-Wolfe
* Lasserre hierarchies

## Disclaimer

This package is under active development. Tests and documentation still
have to be written.
